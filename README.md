# Blu Masonry - Website Oficial

Este é o repositório oficial do website da **Blu Masonry**, projetado, desenvolvido e estruturado pela **Agência StudioC**. 

O site foi construído utilizando tecnologias modernas de desenvolvimento web para garantir excelente performance, pontuação de SEO elevada e total responsividade.

---

## 🚀 Tecnologias Utilizadas

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
* **Animações:** [Framer Motion](https://www.framer.com/motion/) / CSS Customizado
* **Otimização de Imagens:** Suporte nativo do Next.js com compressão automática em formatos modernos (WebP).

---

## 📁 Estrutura do Projeto

* `/app` - Páginas, rotas e estilos globais (inclui páginas dinâmicas para áreas de serviço e serviços individuais).
* `/components` - Componentes reutilizáveis de interface (menus, rodapés, seções interativas, etc.).
* `/public` - Mídias oficiais do site (imagens da galeria de antes/depois, logos oficiais, depoimentos de clientes e vídeos de apresentação).
* `/lib` - Funções utilitárias e classes auxiliares.

---

## 🛠️ Como Executar o Projeto Localmente

### Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em seu computador (versão 18.x ou superior recomendada).

### 1. Clonar ou Extrair o Código
Se for via Git:
```bash
git clone https://github.com/Blumasonry/blumasonry-site.git
cd blumasonry-site
```

### 2. Instalar as Dependências
Instale todos os pacotes necessários na raiz do projeto:
```bash
npm install
```

### 3. Iniciar o Servidor de Desenvolvimento
Inicie o servidor localmente:
```bash
npm run dev
```
Abra o navegador em [http://localhost:3000](http://localhost:3000) para visualizar o site ativo.

---

## 📦 Compilação para Produção

Para gerar uma versão otimizada do site pronta para subir em servidores de produção:
```bash
npm run build
```
